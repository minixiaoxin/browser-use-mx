## 🚀 快速开始

### 方式一：使用 YAML 任务文件（推荐用于测试）

1. **创建任务文件** `tests/agent_tasks/my_task.yaml`：

```yaml
name: 我的第一个任务
task: 在百度搜索"Python教程"，返回前3个搜索结果的标题
judge_context:
  - 必须访问百度搜索
  - 必须搜索"Python教程"
  - 必须返回至少3个搜索结果标题
max_steps: 10
```

2. **执行任务**：

```bash
# 执行所有任务
uv run python tests/ci/evaluate_tasks.py

# 执行单个任务
uv run python tests/ci/evaluate_tasks.py --task tests/agent_tasks/my_task.yaml
```

### 方式二：直接编写 Python 脚本（推荐用于生产）

创建 `my_automation.py`：

```python
from browser_use import Agent, ChatBrowserUse
from dotenv import load_dotenv
import asyncio

load_dotenv()

async def main():
    agent = Agent(
        task='在百度搜索"Python教程"，返回前3个搜索结果的标题',
        llm=ChatBrowserUse(),
    )
    
    history = await agent.run(max_steps=10)
    
    # 获取结果
    result = history.final_result()
    print(f"\n✅ 任务完成！\n结果：{result}")
    
    # 查看访问的URL
    print(f"\n访问的网址：{history.urls()}")

if __name__ == "__main__":
    asyncio.run(main())
```

运行：
```bash
uv run python my_automation.py
```

## 📝 任务文件格式

### YAML 格式说明

```yaml
name: 任务名称（必填）
  # 简短描述任务的名称

task: 任务描述（必填）
  # 详细描述 Agent 需要完成的任务
  # 可以是多行文本，越详细越好

judge_context: （可选）
  # 评判任务是否成功的标准列表
  - 标准1：必须完成某个操作
  - 标准2：必须返回特定信息
  - 标准3：必须访问特定网站

max_steps: 10 （可选，默认15）
  # Agent 最多执行的步骤数
```

### 示例任务文件

**简单搜索任务：**
```yaml
name: 百度搜索
task: 在百度搜索"机器学习"，返回第一个结果的标题和链接
judge_context:
  - 必须访问 baidu.com
  - 必须搜索"机器学习"
  - 必须返回标题和链接
max_steps: 8
```

**电商任务：**
```yaml
name: 淘宝商品搜索
task: |
  1. 访问淘宝网
  2. 搜索"机械键盘"
  3. 提取前5个商品的名称和价格
  4. 按价格从低到高排序
judge_context:
  - 必须访问淘宝网站
  - 必须搜索"机械键盘"
  - 必须返回至少5个商品信息
  - 必须包含商品名称和价格
  - 必须按价格排序
max_steps: 15
```

**表单填写任务：**
```yaml

  访问 https://example.com/contact
  填写联系表单：
  - 姓名：张三
  - 邮箱：zhangsan@example.com
  - 消息：这是一条测试消息

  - 必须访问指定网址
  - 必须填写所有字段
  - 必须提交表单
```

## 🎯 执行任务的三种方式

### 1. 使用评估脚本（适合批量测试）

```bash
# 执行所有任务（并行执行，最多10个同时运行）
uv run python tests/ci/evaluate_tasks.py

# 执行指定目录的任务
uv run python tests/ci/evaluate_tasks.py path/to/tasks/

# 执行单个任务
uv run python tests/ci/evaluate_tasks.py --task tests/agent_tasks/my_task.yaml
```

**特点：**
- ✅ 自动评判任务是否成功
- ✅ 支持并行执行多个任务
- ✅ 生成详细的测试报告
- ✅ 适合 CI/CD 集成

### 2. 直接编写 Python 脚本（适合生产环境）

```python
from browser_use import Agent, Browser, ChatBrowserUse
from dotenv import load_dotenv
import asyncio

load_dotenv()

async def my_task():
    # 自定义浏览器配置
    browser = Browser(
        headless=False,  # 显示浏览器窗口
        window_size={'width': 1920, 'height': 1080},
    )
    
    agent = Agent(
        task='你的任务描述',
        llm=ChatBrowserUse(),
        browser=browser,
    )
    
    history = await agent.run(max_steps=10)
    
    # 处理结果
    if history.is_done():
        print("✅ 任务成功完成！")
        print(f"结果：{history.final_result()}")
    else:
        print("❌ 任务未完成")
        print(f"错误：{history.errors()}")
    
    return history

if __name__ == "__main__":
    asyncio.run(my_task())
```

**特点：**
- ✅ 完全控制执行流程
- ✅ 可以自定义浏览器配置
- ✅ 可以添加自定义工具
- ✅ 适合复杂业务逻辑

### 3. 使用同步方式（最简单）

```python
from browser_use import Agent, ChatBrowserUse
from dotenv import load_dotenv

load_dotenv()

agent = Agent(
    task='在GitHub搜索browser-use项目，返回星标数',
    llm=ChatBrowserUse(),
)

# 同步执行（自动处理 asyncio）
history = agent.run_sync()

print(f"结果：{history.final_result()}")
```

**特点：**
- ✅ 最简单的使用方式
- ✅ 不需要处理 async/await
- ✅ 适合快速测试和脚本

### 示例2：创建一个GitHub信息提取脚本

**创建文件** `github_info.py`

```python
from browser_use import Agent, ChatBrowserUse
from dotenv import load_dotenv
import asyncio

load_dotenv()

async def get_github_info(repo_name: str):
    """获取GitHub仓库信息"""
    agent = Agent(
        task=f'''
        访问GitHub，搜索"{repo_name}"仓库
        提取以下信息：
        1. 仓库星标数
        2. Fork数
        3. 最新发布版本
        4. 主要编程语言
        ''',
        llm=ChatBrowserUse(),
    )
    
    history = await agent.run(max_steps=10)
    return history.final_result()

async def main():
    repos = ['browser-use', 'playwright', 'selenium']
    
    for repo in repos:
        print(f"\n{'='*50}")
        print(f"正在获取 {repo} 的信息...")
        print(f"{'='*50}")
        
        result = await get_github_info(repo)
        print(f"\n结果：\n{result}\n")

if __name__ == "__main__":
    asyncio.run(main())
```

**运行：**
```bash
uv run python github_info.py
```

### 示例3：创建一个带自定义工具的任务

```python
from browser_use import Agent, Browser, Tools, ChatBrowserUse, ActionResult
from dotenv import load_dotenv
import asyncio

load_dotenv()

# 创建自定义工具
tools = Tools()

@tools.action('保存数据到文件')
def save_to_file(data: str, filename: str) -> ActionResult:
    """保存提取的数据到文件"""
    with open(filename, 'w', encoding='utf-8') as f:
        f.write(data)
    return ActionResult(
        extracted_content=f"数据已保存到 {filename}",
        success=True
    )

@tools.action('询问用户')
def ask_user(question: str) -> str:
    """在需要时询问用户"""
    answer = input(f"\n❓ {question}\n👉 ")
    return f"用户回答：{answer}"

async def main():
    agent = Agent(
        task='''
        1. 在百度搜索"Python最佳实践"
        2. 提取前5个搜索结果
        3. 使用 save_to_file 工具保存到 results.txt
        ''',
        llm=ChatBrowserUse(),
        tools=tools,
    )
    
    history = await agent.run(max_steps=15)
    print(f"\n✅ 任务完成！结果：{history.final_result()}")

if __name__ == "__main__":
    asyncio.run(main())
```

## 🎓 最佳实践

### 1. 任务描述要清晰具体

❌ **不好的描述：**
```yaml
task: 搜索一些信息
```

✅ **好的描述：**
```yaml
task: |
  1. 访问百度搜索
  2. 搜索关键词"Python教程"
  3. 提取前5个搜索结果的标题和链接
  4. 返回格式化的列表
```

### 2. 设置合理的步骤数

- 简单搜索任务：5-10 步
- 表单填写任务：8-12 步
- 复杂数据提取：15-20 步
- 多页面操作：20-30 步

### 3. 使用判断标准确保质量

```yaml
judge_context:
  - 必须访问正确的网站
  - 必须完成所有指定操作
  - 返回的数据必须包含所有必需字段
  - 数据格式必须正确
```

### 4. 处理错误和异常

```python
async def safe_task():
    try:
        agent = Agent(
            task='你的任务',
            llm=ChatBrowserUse(),
        )
        history = await agent.run(max_steps=10)
        
        if history.is_done():
            return history.final_result()
        else:
            print(f"任务未完成，错误：{history.errors()}")
            return None
            
    except Exception as e:
        print(f"执行出错：{e}")
        return None
```

### 5. 使用云端浏览器绕过限制

```python
browser = Browser(
    use_cloud=True,  # 使用云端浏览器
    cloud_proxy_country_code='us',  # 使用美国代理
)

agent = Agent(
    task='你的任务',
    browser=browser,
    llm=ChatBrowserUse(),
)
```
