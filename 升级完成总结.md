### 第4步：创建自己的任务（30分钟）

**方式A：创建YAML文件**
```yaml
# tests/agent_tasks/my_task.yaml
name: 我的第一个任务
task: 在百度搜索"机器学习"，返回前5个结果
judge_context:
  - 必须访问百度
  - 必须搜索"机器学习"
  - 必须返回5个结果
max_steps: 10
```

执行：
```bash
uv run python run_tasks.py tests/agent_tasks/my_task.yaml
```

**方式B：创建Python脚本**
```python
# my_task.py
from browser_use import Agent, ChatBrowserUse
from dotenv import load_dotenv
import asyncio

load_dotenv()

async def main():
    agent = Agent(
        task='在百度搜索"机器学习"，返回前5个结果',
        llm=ChatBrowserUse(),
    )
    history = await agent.run(max_steps=10)
    print(f"结果：{history.final_result()}")

if __name__ == "__main__":
    asyncio.run(main())
```

执行：
```bash
uv run python my_task.py
```

## 💡 使用技巧

### 1. 选择合适的执行方式

| 场景 | 推荐方式 | 命令 |
|------|---------|------|
| 快速测试 | YAML + 执行器 | `uv run python run_tasks.py task.yaml` |
| 生产环境 | Python脚本 | `uv run python my_script.py` |
| 批量任务 | 评估脚本 | `uv run python tests/ci/evaluate_tasks.py` |
| 简单脚本 | 同步方式 | `agent.run_sync()` |

### 2. 浏览器配置建议

**开发调试时：**
```python
browser = Browser(
    headless=False,  # 显示窗口，方便观察
    window_size={'width': 1920, 'height': 1080},
)
```

**生产环境：**
```python
browser = Browser(
    headless=True,  # 隐藏窗口，提高性能
    use_cloud=True,  # 使用云端浏览器
)
```

**需要绕过限制：**
```python
browser = Browser(
    use_cloud=True,
    cloud_proxy_country_code='us',  # 使用代理
)
```

### 3. 任务描述技巧

❌ **不好的描述：**
```yaml
task: 搜索一些东西
```

✅ **好的描述：**
```yaml
task: |
  1. 访问百度搜索 (baidu.com)
  2. 搜索关键词"Python教程"
  3. 提取前5个搜索结果的标题和链接
  4. 以列表格式返回
```

### 4. 步骤数设置建议

- 简单搜索：5-10步
- 表单填写：8-12步
- 数据提取：15-20步
- 复杂流程：20-30步

## 🐛 常见问题解决

### 1. 浏览器无法启动

**问题：** `Browser failed to start`

**解决：**
```bash
uvx browser-use install
```

### 2. API Key 错误

**问题：** `API key not found`

**解决：** 检查 `.env` 文件
```
BROWSER_USE_API_KEY=your_key_here
```

### 3. 任务超时

**问题：** 任务未在规定步骤内完成

**解决：** 增加步骤数
```python
history = await agent.run(max_steps=20)
```

### 4. 网站访问被拒

**问题：** `Access denied` 或验证码

**解决：** 使用云端浏览器
```python
browser = Browser(use_cloud=True)
```

### 5. 中文乱码

**问题：** 输出中文显示乱码

**解决：** 确保文件编码为 UTF-8
```python
with open('file.txt', 'w', encoding='utf-8') as f:
    f.write(content)
```
