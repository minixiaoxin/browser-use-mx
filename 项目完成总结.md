## 🎯 核心功能

### 1. 三种创建任务的方式

#### 方式1：YAML文件（最简单）
```yaml
name: 我的任务
task: 在百度搜索"Python教程"
max_steps: 10
```

#### 方式2：Python脚本（最灵活）
```python
from browser_use import Agent, ChatBrowserUse
agent = Agent(task='搜索任务', llm=ChatBrowserUse())
history = await agent.run()
```

#### 方式3：同步方式（最快速）
```python
agent = Agent(task='搜索任务', llm=ChatBrowserUse())
history = agent.run_sync()
```

---

### 2. 三种授权方式

#### 方式1：真实浏览器（本地开发推荐）
```python
browser = Browser(
    executable_path='Chrome路径',
    user_data_dir='用户数据目录',
    profile_directory='Default',
)
```

#### 方式2：保存Cookie（生产环境推荐）
```python
browser = Browser(
    user_data_dir='./browser_data',
)
```

#### 方式3：云端配置（最简单）
```python
browser = Browser(
    use_cloud=True,
    cloud_profile_id='your-profile-id',
)
```

---

### 3. 四种任务行为控制方案

#### 方案1：自定义确认工具（推荐）
```python
@tools.action('等待用户确认')
def wait_confirm(desc: str) -> ActionResult:
    choice = input("确认？(y/n): ")
    return ActionResult(success=(choice == 'y'))
```

#### 方案2：限制可用工具
```python
tools = Tools(exclude_actions=['click'])
```

#### 方案3：任务描述控制
```yaml
task: |
  填写表单，但不要提交
  严格禁止点击提交按钮
```

#### 方案4：回调函数（高级）
监听并拦截关键操作

## 🎨 实用示例

### 示例1：百度搜索（基础）
```python
from browser_use import Agent, ChatBrowserUse
import asyncio

async def search():
    agent = Agent(
        task='在百度搜索"Python教程"，返回前5个结果',
        llm=ChatBrowserUse(),
    )
    history = await agent.run(max_steps=10)
    print(history.final_result())

asyncio.run(search())
```

---

### 示例2：淘宝搜索（带授权）
```python
from browser_use import Agent, Browser, ChatBrowserUse
import asyncio

async def taobao_search():
    browser = Browser(
        user_data_dir='./taobao_data',  # 使用保存的登录状态
        headless=False,
    )
    
    agent = Agent(
        task='在淘宝搜索"机械键盘"，返回前5个商品',
        llm=ChatBrowserUse(),
        browser=browser,
    )
    
    history = await agent.run(max_steps=15)
    print(history.final_result())

asyncio.run(taobao_search())
```

---

### 示例3：表单填写（带确认）
```python
from browser_use import Agent, Browser, Tools, ChatBrowserUse, ActionResult
import asyncio

tools = Tools()

@tools.action('等待用户确认')
def wait_confirm(desc: str) -> ActionResult:
    print(f"\n⏸️ {desc}")
    choice = input("确认？(y/n): ")
    return ActionResult(success=(choice == 'y'))

async def form_task():
    browser = Browser(headless=False)
    
    agent = Agent(
        task='''
        1. 填写表单
        2. 使用 wait_confirm 工具确认
        3. 只有确认后才能提交
        ''',
        llm=ChatBrowserUse(),
        browser=browser,
        tools=tools,
    )
    
    history = await agent.run(max_steps=20)
    print(history.final_result())

asyncio.run(form_task())
```
