## ⚡ 快速执行命令

### 1. 执行单个YAML任务（最简单）

```bash
# 使用任务执行器（推荐）
uv run python run_tasks.py tests/agent_tasks/baidu_search.yaml

# 使用评估脚本（带评判）
uv run python tests/ci/evaluate_tasks.py --task tests/agent_tasks/baidu_search.yaml
```

### 2. 执行所有YAML任务

```bash
# 使用评估脚本（并行执行，带评判）
uv run python tests/ci/evaluate_tasks.py

# 使用任务执行器（交互式）
uv run python run_tasks.py
# 然后选择选项 3
```

### 3. 执行Python脚本

```bash
# 入门示例
uv run python my_first_automation.py

# 高级示例
uv run python advanced_automation.py

# 原有的简单示例
uv run python examples/simple.py
```

## 🎓 三种创建任务的方式

### 方式1️⃣：YAML任务文件（推荐用于测试）

**创建文件** `tests/agent_tasks/my_task.yaml`：

```yaml
name: 我的任务
task: 在百度搜索"Python"，返回前3个结果
judge_context:
  - 必须访问百度
  - 必须搜索"Python"
  - 必须返回3个结果
max_steps: 10
```

**执行：**
```bash
uv run python run_tasks.py tests/agent_tasks/my_task.yaml
```

**优点：**
- ✅ 简单易懂
- ✅ 自动评判
- ✅ 适合CI/CD

---

### 方式2️⃣：Python脚本（推荐用于生产）

**创建文件** `my_script.py`：

```python
from browser_use import Agent, ChatBrowserUse
from dotenv import load_dotenv
import asyncio

load_dotenv()

async def main():
    agent = Agent(
        task='在百度搜索"Python"，返回前3个结果',
        llm=ChatBrowserUse(),
    )
    
    history = await agent.run(max_steps=10)
    print(f"结果：{history.final_result()}")

if __name__ == "__main__":
    asyncio.run(main())
```

**执行：**
```bash
uv run python my_script.py
```

**优点：**
- ✅ 完全控制
- ✅ 可添加自定义逻辑
- ✅ 适合复杂场景

---

### 方式3️⃣：同步方式（最简单）

**创建文件** `simple_script.py`：

```python
from browser_use import Agent, ChatBrowserUse
from dotenv import load_dotenv

load_dotenv()

agent = Agent(
    task='在百度搜索"Python"，返回前3个结果',
    llm=ChatBrowserUse(),
)

# 同步执行（不需要 async/await）
history = agent.run_sync()
print(f"结果：{history.final_result()}")
```

**执行：**
```bash
uv run python simple_script.py
```

**优点：**
- ✅ 最简单
- ✅ 不需要处理异步
- ✅ 适合快速测试

## 🔧 常用配置

### 显示/隐藏浏览器窗口

**YAML方式：** 在执行时添加参数
```bash
uv run python run_tasks.py tests/agent_tasks/baidu_search.yaml --headless
```

**Python方式：** 修改代码
```python
browser = Browser(
    headless=True,  # True=隐藏，False=显示
)
```

---

### 调整窗口大小

```python
browser = Browser(
    window_size={'width': 1920, 'height': 1080},
)
```

---

### 使用云端浏览器（绕过限制）

```python
browser = Browser(
    use_cloud=True,  # 使用云端浏览器
    cloud_proxy_country_code='us',  # 使用美国代理
)
```

---

### 添加自定义工具

```python
from browser_use import Tools, ActionResult

tools = Tools()

@tools.action('保存到文件')
def save_file(content: str, filename: str) -> ActionResult:
    with open(filename, 'w', encoding='utf-8') as f:
        f.write(content)
    return ActionResult(
        extracted_content=f"已保存到 {filename}",
        success=True
    )

agent = Agent(
    task='你的任务',
    llm=ChatBrowserUse(),
    tools=tools,
)
```

## 📊 查看执行结果

### 获取最终结果
```python
result = history.final_result()
```

```python
urls = history.urls()
```

### 查看执行步骤
```python
steps = history.number_of_steps()
```

### 查看执行时长
```python
duration = history.total_duration_seconds()
```


## 🐛 常见问题

### 1. 浏览器无法启动

**解决方案：**
```bash
# 重新安装 Chromium
uvx browser-use install
```

---

### 2. API Key 未设置

**解决方案：**
检查 `.env` 文件是否包含：
```
BROWSER_USE_API_KEY=your_key_here
```

---

### 3. 任务超时

**解决方案：**
增加最大步骤数：
```yaml
max_steps: 20  # 增加到20步
```

或在Python中：
```python
history = await agent.run(max_steps=20)
```

---

### 4. 网站访问被拒绝

**解决方案：**
使用云端浏览器：
```python
browser = Browser(use_cloud=True)
```
